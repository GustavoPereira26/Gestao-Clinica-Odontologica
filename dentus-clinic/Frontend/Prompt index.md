# Prompt da criação base da tela de login

Estou criando um sistema web para clínica odontológica.  
Essa tela é de Login.  
Use HTML, CSS, JavaScript e Bootstrap dentro do asp.net

## A tela possui:

+ um retângulo dividido ao meio centralizado na pagina

+ ao lado esquerdo do retângulo uma mensagem de boas vindas ao sistema

+ ao lado esquerdo do retângulo, abaixo da mensagem um formulário com campos de email e senha

+ ao lado esquerdo do retângulo, abaixo do formulário, um unico botão de Login com alinhamento centralizado verticalmente com o formulário.

+ ao lado direito do retângulo crie uma caminho para uma imagem que ainda será decidida



## Regras

+ campos obrigatórios

+ validação dos campos

+ ocultar/visualizar campo senha

+ layout Responsivo
